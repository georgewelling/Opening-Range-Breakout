# Opening-Range-Breakout
ORB execution engine with broker constraint handling, session gating and risk managed order placement using MT5.

This system: 
- Dynamically calculates an opening range window (NYSE open aligned to London time)
- Monitors live market conditions
- Places broker-compliant stop orders on confirmed breakout
- Enforces structured risk-to-reward constraints
- Applies session gating to prevent overtrading

This project demonstrates experience with: 
- API integration (MT5)
- Event-driven execution systems
- Timezone-aware financial data handling
- Order lifecycle management
- Risk modellling and constraint enforcement
- Deffensive programming for live systems
- Structured logging and monitoring
